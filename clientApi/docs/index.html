This is code documentation of nodejs API of star navigation system


Following is the list of all the apis and brief description

  • /help : Informs about all the apis available in this nodejs app at this port.
  • /getEvents/:packet_type : To get the events based on packet_type send in api get request.
  • /getEventParams/pid :To get the EventParams based on pid in api request
  • /getFlights/:type/:paramValue: To get the flight based on either asset or date
  • /getFlightForAsset/:assetid/:dates : To get flight for a given asset
  • /getLatestNFlights/:assetid/:limit : To getLatestNFlights for provided assetid.
  • /getFlightDetails/flightid : To get flight data based on flightid /aircraftid passed.
  • /saveGeoFence: To insert geofence data in database.
